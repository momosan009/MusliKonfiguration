
# CrazyMüsli Konfigurator

## Beschreibung
Web-Technologien Projekt: Ein Müsli-Konfigurator, bei dem Nutzer ihr eigenes Müsli zusammenstellen können.

## Funktionen
- Registrierung und Login
- 3-stufiger Konfigurator
- 20+ Zutaten auswählbar
- Live-Preisberechnung
- Visuelle Darstellung der Auswahl
- Speichern von Konfigurationen
- Gutscheincode-System
- Vorkonfigurierte Produkte
- Suche und Filter

## Technologien
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Installation
1. Projekt in htdocs kopieren
2. XAMPP starten
3. Datenbank importieren (sql/schema.sql)
4. db.php konfigurieren
5. Projekt im Browser öffnen
